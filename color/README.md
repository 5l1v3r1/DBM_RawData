# Color Command
Get color info from HEX, RGB, name, or image url!

Usage: [p]color (hex, rgb, name, imgURL)  
Correct Formats:  
`#00FF00` OR `00FF00`  
`blue`  
`rgb(135, 206, 235)` OR `rgb(135,206,235)`  
`(image URL)`  

# Node Modules
This requires a few node modules:
- is-image-url
- color-thief-node
- rgb-hex
- color-convert
